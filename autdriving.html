Autonomous Driving in Volkswagen Golf GTI

This project was to develop controllers for a Volkswagen Golf GTI to autonomously drive an oval path. The controller simulation profile was determined by the following steps:

  1. Start with a longitudinal lookahead controller and implement speed control on a straight road 
  2. Tune controller gains according to desired longitudinal and lateral acceleration, lateral error, and heading error 
  3. Combine speed profile and longitudinal controller with lateral control law for full steering control in given path
  4. Simulate the developed controllers in a detailed simulation setting that incorporates more realistic actuator dynamics and sensor noise
  5. Repeat above steps with a PID controller
The gains were chosen to result stable driving in Volkswagen Golf GTI. Although unable to test the code on the car due to COVID restrictions, following plots show the simulation results and that both controllers were albe to result a stable system and ready to test in a Volkswagen car.
